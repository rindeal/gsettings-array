## Quick Start

Install using pip:

```bash
pip install gsettings-array
```

Basic usage:

```bash
gsettings-array COMMAND SCHEMA KEY [OPTIONS]
```

For more detailed information, check out our [full documentation](https://rindeal.github.io/gsettings-array/).
